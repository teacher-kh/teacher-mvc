# Teacher MVC Project

This is a simple MVC (Model-View-Controller) implementation for a teacher management system.

## Project Structure

- `config/` - Configuration files
- `core/` - Core MVC classes
- `app/` - Application specific files
  - `controllers/` - Controller classes
  - `models/` - Model classes
  - `views/` - View files
- `public/` - Publicly accessible files

## Setup

1. Clone the repository
2. Configure your web server to point to the `public/` directory
3. Copy `config/config.example.php` to `config/config.php` and update with your settings
4. Import the database schema from `database/schema.sql`

## Usage

Access the application through your web browser. The default route is set to the Home controller.
